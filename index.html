Rendering

Questions: Rendering quiz

Home assignment:
  1. Study the coding demo simple-rendering.js
  2. Make changes, be creative and make it look cool ;)
  3. Change 3D cube model with any other mesh using Blender WebGL export addon, check blender/export_webgl.py
  4. Change object and camera transformations inside draw() function
  5. Change colors using bgColor and fgColor variables
  6. Distort object shape inside vertexShader
  7. Distort object colors inside fragmentShader

GPU

Home assignment: Find out what GPU models are inside of all your devices
  1. Mobile phone
  2. Tablet
  3. Laptop
  4. PC
  5. Gaming console
Figure out for each GPU you find
  1. Is it integrated or dedicated?
  2. What kinds of graphics APIs it supports?
  3. What kinds of shading languages it supports?
Create and commit a file containing the anwers: mygpu.html
